Trees

Explore different types of tree data structures

Binary Tree

A hierarchical data structure where each node has at most two children.

Use Cases:

Expression trees, decision trees, hierarchical data

Operations:

Insert node, Delete node, Tree traversal

Example:
10 → [5, 15]

Binary Search Tree

A binary tree where left subtree values are smaller and right subtree values are larger.

Use Cases:

Fast searching, sorted data, database indexing

Operations:

Insert, Delete, Search

Example:
Inorder → 5 → 7 → 10 → 15

AVL Tree

A self-balancing BST where height difference is at most 1.

Use Cases:

Balanced search operations, dynamic datasets

Operations:

Rotations, Insert, Delete

Example:
Insert 10 → 20 → 30 → Rotation
