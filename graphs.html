Graphs

Explore different graph traversal and analysis algorithms

Breadth First Search (BFS)

BFS explores nodes level by level using a queue.

Use Cases:

Shortest path in unweighted graphs, social networks, web crawling

Operations:

Queue-based traversal, level-order visiting

Example:
A → B → C → D

Depth First Search (DFS)

DFS explores as deep as possible before backtracking.

Use Cases:

Cycle detection, topological sorting, maze solving

Operations:

Recursive / stack-based traversal

Example:
A → B → C → Backtrack

Cycle Detection

Detects whether a graph contains a cycle.

Use Cases:

Deadlock detection, circuit design, dependency graphs

Operations:

DFS recursion stack / Union-Find

Example:
A → B → C → A (cycle)
